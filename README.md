*   **Overview:** Automated auditor executing 12 CIS benchmark controls across multi-cloud configurations (AWS, Azure, GCP).

*   **Tech Stack:** Python, AWS CLI, Azure CLI, GCP SDK, Pytest, GitHub Actions

*   **Tech Stack:** Bash, Nmap, Curl, Linux

### 💻 Systems & Network Projects

*   **Overview:** High-performance cross-platform file manager written in C99 with zero external library dependencies (163 KB binary).
